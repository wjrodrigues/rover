# Rover

Aplicação que orienta o Rover em uma planície

## Iniciar aplicação

Inicia os containers

```bash
make start
```

Acompanhar status container

```bash
docker logs rover_app -f
```

## Documentação
Atualiza documentação da aplicação

```bash
make update-doc
```

Link da documentação: http://localhost:3000/api-docs
